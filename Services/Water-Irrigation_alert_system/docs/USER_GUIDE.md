## For Farmers (किसानों के लिए)

### What is KisanVoice? (किसानवॉइस क्या है?)

KisanVoice is a smart irrigation system that sends you SMS alerts telling you when to water your crops and warns you about dangerous weather conditions.

किसानवॉइस एक स्मार्ट सिंचाई प्रणाली है जो आपको SMS भेजती है कि कब फसल को पानी देना है और खतरनाक मौसम के बारे में चेतावनी देती है।

### Benefits (लाभ)

✅ Save 30-40% water (30-40% पानी बचाएं)  
✅ Reduce irrigation costs (सिंचाई खर्च कम करें)  
✅ Increase crop yield (फसल उत्पादन बढ़ाएं)  
✅ Get weather warnings (मौसम की चेतावनी पाएं)  
✅ Track water savings (पानी की बचत ट्रैक करें)

### How to Register (रजिस्ट्रेशन कैसे करें)

**Required Information (जरूरी जानकारी):**
1. Mobile number (मोबाइल नंबर)
2. Name (नाम)
3. Crop type (फसल का प्रकार)
4. Sowing date (बुवाई की तारीख)
5. Farm area in acres (खेत का क्षेत्रफल एकड़ में)
6. District (जिला)
7. GPS location (optional) (GPS लोकेशन - वैकल्पिक)

**Steps (कदम):**
1. Open mobile app (मोबाइल ऐप खोलें)
2. Click "Register" (रजिस्टर पर क्लिक करें)
3. Fill all details (सभी जानकारी भरें)
4. Submit (सबमिट करें)
5. You'll receive confirmation SMS (आपको पुष्टि SMS मिलेगा)

### Daily Irrigation Alerts (रोज की सिंचाई सलाह)

**When:** Every day at 5:45 PM IST (हर दिन शाम 5:45 बजे)

**Message Types (संदेश के प्रकार):**

**1. Irrigate (सिंचाई करें)**
```
[KisanVoice] 🌾 सिंचाई सलाह
राजेश जी, गेहूं
कल सुबह 6-8 बजे सिंचाई करें

मौसम: 28°C, साफ
मिट्टी: 45% नमी
पानी बचत: 2,500 लीटर
```

**2. Skip (छोड़ें)**
```
[KisanVoice] 💧 सिंचाई रोकें
राजेश जी, गेहूं
आज सिंचाई न करें

मौसम: 26°C, बारिश संभव
बारिश: 15mm (अगले 2 दिन)
पानी बचत: 3,000 लीटर
```

**3. Recovery (वापसी)**
```
[KisanVoice] ⚠️ सिंचाई करें
राजेश जी, गेहूं
बारिश नहीं हुई, आज सिंचाई करें

मौसम: 30°C, साफ
मिट्टी: 35% नमी (कम)
```

### Weather Alerts (मौसम की चेतावनी)

**When:** Checked every 3 hours, sent when needed (हर 3 घंटे में जांच, जरूरत पर भेजा जाता है)

**Alert Types (चेतावनी के प्रकार):**

**1. Heatwave (गर्मी) - Above 40°C**
```
[KisanVoice] ⚠️ गर्मी की चेतावनी
गेहूं खतरे में! तापमान: 42°C

सुबह/शाम सिंचाई करें
दोपहर में काम न करें
```

**2. Frost (पाला) - Below 5°C**
```
[KisanVoice] ❄️ पाला चेतावनी
गेहूं खतरे में! तापमान: 3°C

रात में हल्की सिंचाई करें
फसल ढकें
```

**3. Thunderstorm (आंधी)**
```
[KisanVoice] ⛈️ आंधी चेतावनी
आंधी आ रही है!

खेत में न जाएं
उपकरण सुरक्षित करें
```

**4. Heavy Rain (भारी बारिश) - Above 50mm**
```
[KisanVoice] 🌧️ भारी बारिश चेतावनी
50mm+ बारिश संभव

जल निकासी जांचें
सिंचाई रोकें
```

**5. High Wind (तेज हवा) - Above 40 km/h**
```
[KisanVoice] 💨 तेज हवा चेतावनी
हवा: 45 km/h

उपकरण बांधें
छिड़काव न करें
```

**6. Drought (सूखा) - 10+ days no rain**
```
[KisanVoice] 🏜️ सूखा चेतावनी
10 दिन से बारिश नहीं

तुरंत सिंचाई करें
मल्चिंग करें
```

### Best Practices (सर्वोत्तम प्रथाएं)

**DO (करें):**
- ✅ Read SMS alerts daily (रोज SMS पढ़ें)
- ✅ Follow irrigation advice (सिंचाई सलाह मानें)
- ✅ Irrigate early morning (6-8 AM) (सुबह जल्दी सिंचाई करें)
- ✅ Check soil moisture (मिट्टी की नमी जांचें)
- ✅ Provide GPS location for accuracy (सटीकता के लिए GPS दें)

**DON'T (न करें):**
- ❌ Ignore SMS alerts (SMS को नजरअंदाज न करें)
- ❌ Irrigate at noon (दोपहर में सिंचाई न करें)
- ❌ Over-water crops (फसल को ज्यादा पानी न दें)
- ❌ Irrigate immediately after rain (बारिश के तुरंत बाद सिंचाई न करें)

### FAQs (सामान्य प्रश्न)

**Q: When will I get SMS? (SMS कब आएगा?)**  
A: Daily at 5:45 PM. Weather alerts anytime. (रोज शाम 5:45 बजे। मौसम की चेतावनी कभी भी।)

**Q: Is SMS free? (क्या SMS मुफ्त है?)**  
A: Yes, completely free. (हां, बिल्कुल मुफ्त।)

**Q: Can I ignore the advice? (क्या मैं सलाह को नजरअंदाज कर सकता हूं?)**  
A: Yes, but following saves water and money. (हां, लेकिन मानने से पानी और पैसे बचते हैं।)

**Q: What if SMS doesn't come? (अगर SMS नहीं आया?)**  
A: Check phone network. Contact support. (फोन नेटवर्क जांचें। सपोर्ट से संपर्क करें।)

**Q: Is GPS required? (क्या GPS जरूरी है?)**  
A: No, but GPS gives more accurate alerts. (नहीं, लेकिन GPS से ज्यादा सटीक सलाह मिलती है।)

**Q: How to stop SMS? (SMS कैसे बंद करें?)**  
A: Reply "STOP" or unregister in app. ("STOP" भेजें या ऐप में अनरजिस्टर करें।)

## For Developers

### Quick Start

**1. Clone Repository**
```bash
git clone https://github.com/your-org/kisanvoice-irrigation.git
cd kisanvoice-irrigation
```

**2. Install Dependencies**
```bash
npm install
```

**3. Configure AWS**
```bash
aws configure
# Region: ap-south-1
```

**4. Create Secrets**
```bash
aws secretsmanager create-secret \
  --name kisanvoice/prod \
  --secret-string '{
    "TWILIO_ACCOUNT_SID": "your_sid",
    "TWILIO_AUTH_TOKEN": "your_token",
    "TWILIO_MESSAGING_SERVICE_SID": "your_service_sid",
    "OPENWEATHER_API_KEY": "your_api_key"
  }'
```

**5. Deploy**
```bash
npx serverless deploy
```

**6. Seed Data**
```bash
npm run seed:crops
npm run seed:monsoon
```

### React Native Integration

**Install Dependencies:**
```bash
npm install axios @react-native-async-storage/async-storage
```

**API Service:**
```javascript
import axios from 'axios';

const API_BASE_URL = 'https://ys4xa8tu60.execute-api.ap-south-1.amazonaws.com/dev';

export const IrrigationAPI = {
  registerFarmer: async (data) => {
    const response = await axios.post(`${API_BASE_URL}/irrigation/register`, data);
    return response.data;
  },
  
  getDashboard: async (farmerId) => {
    const response = await axios.get(`${API_BASE_URL}/irrigation/dashboard/${farmerId}`);
    return response.data;
  },
  
  getAlertsByPhone: async (phone) => {
    const response = await axios.get(
      `${API_BASE_URL}/irrigation/alerts/phone/${encodeURIComponent(phone)}`
    );
    return response.data;
  },
  
  deleteAlert: async (farmerId, alertId) => {
    const response = await axios.delete(
      `${API_BASE_URL}/irrigation/alerts/delete/${farmerId}/${encodeURIComponent(alertId)}`
    );
    return response.data;
  },
  
  getCropCalendar: async (crop) => {
    const response = await axios.get(`${API_BASE_URL}/irrigation/crop-calendar/${crop}`);
    return response.data;
  }
};
```

**Usage Example:**
```javascript
import { IrrigationAPI } from './services/irrigationAPI';

// Register farmer
const registerFarmer = async () => {
  try {
    const result = await IrrigationAPI.registerFarmer({
      phone: '+919876543210',
      name: 'Ramesh Kumar',
      crop: 'wheat',
      cropStage: 'flowering',
      district: 'Karnal',
      lat: 29.6857,
      lon: 76.9905,
      language: 'hi'
    });
    
    // Save farmerId
    await AsyncStorage.setItem('farmerId', result.farmerId);
    console.log('Registration successful:', result);
  } catch (error) {
    console.error('Registration failed:', error);
  }
};

// Get dashboard
const loadDashboard = async () => {
  try {
    const farmerId = await AsyncStorage.getItem('farmerId');
    const dashboard = await IrrigationAPI.getDashboard(farmerId);
    console.log('Dashboard:', dashboard);
  } catch (error) {
    console.error('Failed to load dashboard:', error);
  }
};
```

### Troubleshooting

**Common Issues:**

**1. SMS not delivered**
- Check Twilio credentials in Secrets Manager
- Verify phone number format (E.164)
- Check Twilio account balance
- Review CloudWatch logs

**2. Weather data not fetched**
- Verify OpenWeatherMap API key
- Check API rate limits
- Review error logs

**3. EventBridge rule not triggering**
- Verify rule is enabled
- Check rule schedule expression
- Review Lambda permissions

**4. DynamoDB errors**
- Check IAM permissions
- Verify table names
- Review capacity settings
