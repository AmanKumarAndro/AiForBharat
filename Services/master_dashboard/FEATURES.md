# Features

## Core Features

### 1. Dashboard Overview
Aggregated view of all platform metrics including total farmers, active services, and system health.

### 2. Voice AI Analytics
- Track voice AI session usage
- Monitor top questions asked by farmers
- Analyze average latency and performance metrics
- Session history and trends

### 3. Helping Hand Services
Complete service request management system:
- Service request tracking and status monitoring
- Provider directory management
- Treatment database for crop diseases
- Banned pesticides registry
- KVK (Krishi Vigyan Kendra) contact directory
- Pincode-based service mapping

### 4. Irrigation Management
Smart irrigation system monitoring:
- Farmer enrollment tracking
- Crop data management
- Monsoon calendar integration
- Water savings calculations
- SMS notification logs
- Soil moisture state monitoring

### 5. Farmer Analytics
- Geographic distribution by state and city
- Language preference analysis
- Land area statistics
- Profile completion tracking
- Multi-source user aggregation

### 6. Activity Monitoring
Real-time activity feed showing:
- Recent voice queries
- Service requests
- System events
- User interactions

### 7. Alert System
Centralized alert management:
- Weather alerts
- Irrigation alerts
- Critical notifications

## Data Sources

The API aggregates data from 15+ DynamoDB tables:
- Authentication & user data
- Voice AI sessions
- Service requests & providers
- Irrigation system data
- Treatment & pesticide databases
- Geographic mappings

## Key Capabilities

- Real-time data aggregation
- Multi-table query optimization
- Automatic data deduplication
- Geographic analytics
- Revenue tracking
- Performance monitoring
